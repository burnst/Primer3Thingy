Application Bootstrap

This page will bootstrap the state of the database by loading in lookup data and other instances that are important to get the dispatch application started and up and running so that you can log in and use, test, and enhance the application.

This expects you to have mongodb installed and running on your development box. It also expects to be able to contact the campus IDW during account setup, so make sure you hooked up to the VPN if you are running remotely. The following database objects will be setup by this controller.

  1. the "p3thingy" database will be deleted!
  2. a "p3thingy" database will be created
